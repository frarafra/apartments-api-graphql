# GraphQL API

API using GraphQL, implemented with Apollo GraphQL, MongoDB Atlas and Mongoose. Tests developed using Jest and Supertest.


## Project setup
```bash
	npm install
```

## Run server
```bash
	npm start
```

## Run tests
```bash
	npm test
```

## Queries and mutations
Playground
```bash
http://localhost:4000/
```
